No.297 Serialize and Deserialize Binary Tree

https://leetcode.com/problems/serialize-and-deserialize-binary-tree/

analysis

Use preorder traversal to serialize the data and use recursion to get the preorder traversal. For deserialization, we use queue structure to implement. Since we store data in preorder, we can also reconstruct the binary tree in preorder.

The running time is $O(n)$. The space complexity is $O(n)$.

No.449 Serialize and Deserialize BST

https://leetcode.com/problems/serialize-and-deserialize-bst/

analysis

It can be solved the same as No.297 Serialize and Deserialize Binary Tree.

The main difference is as compact as possible. The properties of a BST: The left (right) subtree of a node contains only nodes with keys smaller (greater) than the node’s key. With the aforementioned BST property, one can actually recover a BST from its preorder traversal alone (without specifying the null nodes).

No.428 Serialize and Deserialize N-ary Tree

https://leetcode.com/problems/serialize-and-deserialize-n-ary-tree/

analysis

Store the data and node relationship in the data strucure like $1[3[5[]6[]]2[]4[]]$ for serialization. And for deserialization, we use stack. We store new node into stack. As long as we find its parent, we add it into its parents’ children array and pop it out.

The time and space complexity would be both $O(n)$.
